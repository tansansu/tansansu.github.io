MLBPARKER 사용 가이드

    1. 로그인

    1-1. 자동 로그인

    자동로그인 기능을 활성화하면 앱을 다시 시작할 때 로그인된 상태로 시작할 수 있습니다.

    • 사이드 메뉴에서 로그인창을 띄워서 자동로그인 항목을 체크하여 활성화
    • 자동로그인을 해제할 경우에는 마이메뉴에 들어가서 로그아웃 후에 다시 로그인 창에서 자동로그인 항목 체크해제
    1-2. 소셜 로그인

    카카오, 네이버 로그인을 통한 소셜로그인 기능은 향후 제공 예정입니다.

    2. 유저 메뉴

    2-1. 유저 차단

    유저 차단에는 아이디 차단과 슈퍼블로킹 2가지 방법이 있습니다. 아이디 차단은 엠팍사이트에서 제공하는 방법으로 최대 차단가능한 개수가 있습니다. 슈퍼블로킹은 MLBPARKER앱에서만 제공하는 기능으로 차단가능 제한숫자가 없으며, 차단되었다는 표시가 화면에 표시됩니다.

    • 게시물 작성자의 차단은 오른쪽 하단 메뉴버튼을 클릭한 후 차단 메뉴 선택
    • 댓글의 유저차단은 댓글의 아바타를 클릭해서 나오는 메뉴 선택
    2-2. 차단리스트 관리
    • 환경설정 메뉴에서 차단리스트의 삭제가 가능함(슈퍼블로킹만 가능하며, 유저차단은 엠팍 사이트에서 관리 가능)
    2-3. 유저 게시물, 가입일 보기
    • 게시글 작성자: 오른쪽 상단의 게시글 작성자 아바타 클릭 시 표시되는 메뉴에서 선택
    • 댓글 작성자: 댓글 작성자 아바타 클릭 시 표시되는 메뉴에서 선택
    2-4. 유저 메모

    앱에서만 동작하는 기능으로 특정 유저에게 아이디를 기준으로 최대 20자까지의 메모를 작성할 수 있습니다. 작성된 메모는 게시판과 게시글에서 유저 네임 옆에 표시됩니다.

    • 게시글 작성자: 오른쪽 상단의 게시글 작성자 아바타 클릭 시 표시되는 메뉴에서 선택
    • 댓글 작성자: 댓글 작성자 아바타 클릭 시 표시되는 메뉴에서 선택
    • 환경설정 메뉴의 메모 관리에서 메모를 수정하거나 삭제할 수 있음

    3. 키워드 차단

    엠팍사이트에서도 개인설정에서 키워드 차단이 가능하지만 현재 MLBPARKER앱에서 제공하는 키워드 차단은 앱에서 단독으로 동작하는 기능입니다. 키워드가 제목에 포함되어 있을 경우에만 해당 글이 표시되지 않으며, 키워드로 차단되는 글이 생길 경우 게시판 로딩 시에 차단된 건수와 키워드가 표시됩니다.

    그래서 앱을 지우거나 휴대폰을 바꾸면 새로 등록이 필요합니다. 이 부분은 설정파일을 저장하여 적용하는 기능을 향후 추가하려고 합니다.

    3-1. 차단 키워드 관리
    • 환경설정 메뉴에서 키워드 차단 목록 관리에서 키워드 추가와 삭제가 가능함

    4. 말머리 관리

    보기 싫은 말머리를 보지 않거나 보고자 하는 말머리만 있을 경우에는 해당 말머리 설정 기능을 통해 할 수 있습니다.

    4-1. 기본 말머리 관리
    • 환경설정 메뉴에서 말머리 목록 관리에서 필터링할 말머리를 관리할 수 있음
    • 체크해제한 말머리는 게시판에서 표시되지 않습니다.
    4-2. 커스텀 말머리 관리
    • 환경설정 메뉴에서 말머리 목록 관리에서 키워드 추가를 통해 특정 단어를 말머리로 지정할 수 있음(예: nba 등)
    4-3. 말머리 검색
    • MLB야구타운, 한국야구타운, BULLPEN 게시판에서 상단 앱바의 말머리 검색 아이콘을 클릭 시 말머리를 선택하는 화면이 나타나며, 말머리 선택 시 해당 게시판에서 선택한 말머리만 표시됩니다.
    • 게시판별로 말머리를 다르게 설정할 수 있으며, 앱을 재실행 시에는 말머리 검색이 풀립니다.
    • 기본 말머리 중에 설정의 말머리 관리에서 체크해제한 말머리는 말머리 검색에서 표시되지 않습니다.

    5. 게시글 이미지

    5-1. 이미지 확대/축소

    게시글의 이미지를 확대하거나 축소하고 싶을 때에는 게시글 이미지를 누르면 표시되는 화면에서 가능합니다.

    5-2. 이미지 저장

    게시글 이미지를 클릭하여 오른쪽 상단의 저장메뉴를 클릭
