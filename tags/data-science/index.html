Coursera의 1만불 Master Degree

March 8, 2016

2 minute read

얼마 전 Coursera에서 이메일 한통이 왔습니다. 매주 보내는 추천 강의 이메일인가 싶었는데, 설문을 요청하는 메일이였습니다. Coursera에서 강의를 들을 때 처음 시작하는 주와 마지막 주에 설문해달라는 메일이 온적은 있었는데, 지금처럼 아무 강의도 듣지 않을 때 설문 요청 메일이 온 적은 처음이였습니다. 게다가 설문 요청도 강의하는 기관(내지 학교)로부터 온 것이 아니고 Coursera로부터 온 것이였고요. 이런 설문조사는 곧장 휴지통에 버려버리는데 호기심이 생겨서 한 번 해봤습다. 캡쳐하지 못한 것이 아쉬운데 일단 질문은 5개정도 되었던거 같아요.

Hugo를 통해 생성한 페이지를 Github에 간단히 연결하기

February 28, 2016

2 minute read

최근 Github에 올리는 블로그 페이지들을 jekyll에서 Hugo로 바꿨습니다. 정적페이지 생성기로 jekyll 외에 여러가지가 있다는 것을 알게 되었고, Hugo는 우연히 인터넷에서 알게 되어 한 번 시도만 해볼까 했는데 결국에는 블로깅 Tool도 바꿔버렸습니다 ^^; 겉으로 보기에는 당연히도 차이점이 없지만 만족스러운 전환이였습니다. 제가 생각하는 Hogo의 장점은 1. 설치과정이 쉽다. 저는 mac에서 jekyll을 설치하고 사용했는데, jekyll을 설치하는 과정이 복잡하고 번거로웠습니다. 반면에, Hugo는 mac 기준으로 home brew를 설치했다면 brew install hugo 명령어만으로 모든 설치가 끝납니다. 페이지 생성 속도가 빠르다.
